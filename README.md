# 🚀 TaskFlow – Task Management App

TaskFlow is a responsive task management web application built using React and TypeScript. It allows users to manage projects and tasks efficiently with a clean and modern UI.

---

## ✨ Features

* 🔐 Authentication (Login flow)
* 📁 Project listing
* 📋 Task management
* ➕ Add new tasks
* 🔍 Filter tasks by status (Todo / In Progress / Done)
* 🔒 Protected routes
* 🎨 Responsive UI with Tailwind CSS

---

## 🛠 Tech Stack

* React + TypeScript
* Tailwind CSS
* React Router
* Vite

---

## 🚀 Getting Started

```bash
git clone https://github.com/Shashank7368/taskflow-shashank.git
cd taskflow-shashank
npm install
npm run dev
```
